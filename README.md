<https://react-github-battle-2d9f1.firebaseapp.com/>

## Prerequisites
Node and npm

## Installation

1. Check out repository / extract directory
2. `cd` into root of project
3. Run `npm install` from the command line

## Running
### Developer Mode
1. Run `npm start`

### Build for Development
1. Run `npm build`
